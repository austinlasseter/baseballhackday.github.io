  • SportRadar

    Visit this link http://developer.sportradar.com/Baseball_Hack_Day and follow instruction to get Real-Time access to its MLB API.

    1. Register for a Mashery account. Under the ‘How did you find us?’ dropdown, select ‘Other’ and enter "Baseball Hack Day" in the ‘Other’ field below.
    2. Confirm your account by clicking the link in Mashery’s confirmation email.
    3. We’ll email you one universal key - with a 1 million call limit - that lets you access our MLB Trial API. (expires in 30 days)

    Detailed documentation is available at our portal - you can also test out calls using the API Sandbox when signed in.

  • SeatGeek

    SeatGeek is a search engine for tickets to live events, including Major League and Minor League baseball games. They have a powerful, easy-to-use API that allows you to tap into their event catalog to obtain information on events, performers, venues, ticket prices and more.

    There is No Rate Limits, No API Keys, and No Caching Restrictions.

    Go to seatgeek.com/build for information on their API , and to platform.seatgeek.com for API documentation
